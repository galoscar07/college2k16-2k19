Foloseste Distante Focale Mari (Telephoto)

Distanta focala reprezinta un factor important in fotografia de portret (persoane). Ca sa o folosesti in avantajul tau, trebuie sa te familiarizezi cu obiectivele care le ai in posesie.

Vestea buna este ca majoritatea camerelor foto din ziua de azi se comercializeaza cu obiective de kit (standardul sa zicem este binecunoscutul 18-55mm), un obiectiv foarte comun, dar care este o unealta excelenta pentru incepatori.

Pentru persoane, se folosesc “de regula” focale peste 50mm. Ca idee, cu cat este mai “lung” obiectivul cu atat este mai bine.

Capatul tele al obiectivului este ideal pentru fotografia de portret (persoane) deoarce lentilele telephoto compreseaza perpectiva imaginii, separa foarte bine subiectul de fundal (indiferent ca e studio sau natura), si deformarea fata de o distanta focala mai mica este nu este atat de evidenta.

Ca si exemplu, fotografia din stanga este realizata la 200mm si F4 (Canon 70-200 L F2.8).
